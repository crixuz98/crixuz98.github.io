# Crixuz Personal Lab

Portafolio estático enfocado en robótica, inteligencia artificial, automatización y sistemas interactivos.

## Páginas

- `index.html`: laboratorio y presentación principal.
- `archivo.html`: libro técnico interactivo para el histórico de proyectos.

## Estilos

- `css/styles.css`: identidad del laboratorio, animaciones y diseño adaptable.
- `css/archive.css`: diseño del libro y sus páginas técnicas.

## Interacciones

- `js/script.js`: secuencia de arranque, campo neural, navegación y telemetría.
- `js/archive.js`: contenido y navegación del libro de proyectos.

## Publicación

El sitio está preparado para publicarse desde la raíz de la rama `main` mediante GitHub Pages:

**https://crixuz98.github.io**

Los proyectos descritos como `CONCEPTO` o `PLANIFICADO` son ideas futuras y no se presentan como trabajos terminados.
