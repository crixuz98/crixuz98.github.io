# Recursos del Aula

Esta carpeta almacena los archivos descargables del apartado **Aula**.

## Organización recomendada

- Usa nombres cortos, descriptivos y sin espacios: `guia-cinematica-diferencial.pdf`.
- Conserva aquí documentos PDF, presentaciones y archivos de apoyo razonablemente pequeños.
- Para videos pesados, usa un servicio de video y enlaza su URL desde `aula.html`.
- Después de agregar un archivo, crea o actualiza su tarjeta en `aula.html`.

## Ejemplo de enlace

```html
<a class="resource-card" href="assets/docs/guia-cinematica-diferencial.pdf">
    ...
</a>
```

Los archivos subidos al repositorio serán públicos cuando GitHub Pages publique la rama `main`.
